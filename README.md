# Health-Prediction-API
Api for health prediction

## Setup

    npm install
    node app/index.js
    
## Versions

1. node:6.5.*
2. express:4.15.*
3. python:2.7.9

## Style-Guide

### JS

#### Standard (ES8)
##### Install

    npm install standard --global

##### Run

    standard --fix

### Python

#### PEP8
##### Install

    pip install pep8

or for Debian

    apt-get install pep8

#### RUN

    pep8 *.py
